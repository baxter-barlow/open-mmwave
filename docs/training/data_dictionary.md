# Data Dictionary - open_mmwave Rev G

## Core JSON Artifacts
- `data/netlist_revG.json`: IPC net connectivity
- `data/thermal_inputs.json`: Datasheet-derived thermal inputs
- `data/thermal_analysis.json`: Computed junction temps
- `data/impedance_targets.json`: Trace width targets
- `data/component_placement.json`: PnP coordinates
- `data/gerber_analysis.json`: Layer statistics
- `data/drill_analysis.json`: Hole size counts
- `data/odb_nets.json`: ODB netlist mapping
- `data/component_values.json`: Passive values from BOM
- `data/component_values.json`: Values include Description fallback for passives
